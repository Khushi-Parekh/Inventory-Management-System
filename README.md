# 📦 Inventory Management System

A full-stack **Inventory Management System** built to efficiently manage products, track expenses, and visualize business insights using interactive charts.

---

## 🚀 Features

* 📊 Interactive dashboard with charts and analytics
* 📦 Product management (add, update, delete, view)
* 💰 Expense tracking and monitoring
* 📈 Data visualization using charts for better insights
* 👥 User management system
* ⚡ Responsive and modern UI

---

## 🛠 Tech Stack

### Frontend

* Next.js
* React
* TypeScript
* Tailwind CSS

### Backend

* Node.js
* Express.js
* Prisma ORM

### Database

* PostgreSQL

---

## 📊 Data Visualization

The dashboard includes charts to provide insights such as:

* Sales summary
* Expense trends
* Product performance

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```id="n6d1sn"
git clone https://github.com/Khushi-Parekh/Inventory-Management-System.git
cd Inventory-Management-System
```

---

### 2️⃣ Setup Backend

```id="3glr6v"
cd server
npm install
```

Create a `.env` file and add:

```id="k87y2m"
DATABASE_URL=your_postgresql_database_url
PORT=8000
```

Run backend:

```id="v9kw7l"
npm run dev
```

---

### 3️⃣ Setup Frontend

```id="c9h5zx"
cd client
npm install
npm run dev
```
